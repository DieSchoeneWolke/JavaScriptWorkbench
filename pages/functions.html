Array Manipulation

There is an array stored in a variable which will be manipulated by several available methods including slice(), reverse(), push(), pop(), join(), trim() and map().
When you click the button below, an alert box will appear and show you the result. 

            
              function Greeting() {
                alert('Hello!');
            }
            document.querySelector("#btn-1").addEventListener("click", function () {
                alert('Hello World');
                Greeting();
            });
